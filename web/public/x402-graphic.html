What it is
Your vault on Base signs a gasless EIP-3009 USDC transfer. The Gitbank Relayer submits it on-chain and retries the API with the payment proof attached. You pay the API. You get the data. The tx lands on Base Mainnet. You never touch a wallet.
1
Tweet a command at @gitbankbot
2
Bot probes the API, gets HTTP 402 payment challenge
3
Relayer signs EIP-3009 from your vault, pays in USDC
4
API returns data. Bot replies with results + full data link
Commands
AI Search @ExaAILabs
@gitbankbot x402-pay @ExaAILabs "gitbank defi protocols" 0.007 USDC

Smart Money @nansen_ai
@gitbankbot x402-pay @nansen_ai 0.05 USDC
Pays Nansen smart money netflow API. Returns which tokens smart traders are moving into across all chains right now.
Any x402 URL
@gitbankbot x402-pay https://your-api.com/data 0.01 USDC
Works with any x402-compatible endpoint. If it returns HTTP 402 with a valid payment challenge, Gitbank handles the rest.
